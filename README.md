# Google_tradutor
Site que imita o google tradutor, feito com HTML, Tailwind CSS, JavaScript, React, JSON, Node.js e Vite.

Comandos:

npm create vite@latest
- Nome do projeto: tradutor
- Framework: React
- Variante: JavaScript

cd tradutor

npm install

npm install -D tailwindcss@3

npx tailwindcss init -p

npm run dev
